## How to update design diagrams

1. Go to [DrawIO](https://app.diagrams.net/)
2. Open from device, select the .drawio file you want to edit
3. Finish diagram updates
3. Select File > Save and save as XML File (.drawio) to Download, and overwrite the prior .drawio file
4. Select File > Export > PNG..., deselect "Include a copy of my diagram", and overwrite the prior .png file
